# PepSIRF: Peptide-based Serological Immune Response Framework  


### Dependencies
- cmake
- boost

### Compiling/Building
```
mkdir build
cd build
cmake ..
make
```

Alternatively, one can use the included build script.
```
chmod +x build.sh
./build.sh
```
Both of these options will create a ```pep_sirf``` executable in the build directory.
